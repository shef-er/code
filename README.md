# code

## setup

```shell
sudo pacman -S code

# for Code - OSS
git clone git@github.com:shef-er/code.git ~/.config/Code\ -\ OSS/User

# for VSCodium
git clone git@github.com:shef-er/code.git ~/.config/VSCodium/User

# for VS Code
git clone git@github.com:shef-er/code.git ~/.config/Code/User
```
